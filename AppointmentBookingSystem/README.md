
# Appointment Booking System (Clinic / Salon)

ASP.NET Core MVC app for online appointment booking.

## Features
- Customer booking form
- Schedule list view (staff)
- Simple conflict prevention (30-minute slots per service)
- Bootstrap UI
- No database required (in-memory store)

## Run
Open `AppointmentBookingSystem.sln` in Visual Studio 2022 and press **F5**.

Pages:
- `/` Home
- `/Appointments/Create` Book
- `/Appointments` Schedule
